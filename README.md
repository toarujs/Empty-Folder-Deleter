# Empty-Folder-Deleter
这是一个用于删除空文件夹的脚本工具，适用于windows系统
输入需要清理的文件夹路径即可清理空文件夹
如果有需要保留的空文件夹，可以将文件夹路径写入白名单文件“white.txt”

This is a script tool for deleting empty folders, designed for Windows systems.
Enter the path of the folder you want to clean to remove empty folders.
If there are empty folders that need to be preserved, you can write the folder paths into the whitelist file 'white.txt'.
